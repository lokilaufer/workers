## 🛠 Технологии
- Webpack
- Koa (сервер)
- Cache API
- GitHub Actions
- GitHub Pages

## 🚀 Запуск

```bash
# Клонирование
git clone git@github.com:lokilaufer/workers.git
cd workers

# Установка зависимостей
npm install

# Запуск фронтенда
npm start

# Запуск сервера (в другом терминале)
npm run server
